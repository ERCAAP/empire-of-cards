2. Oyun Mekanikleri

Core Loop

HAZIRLIK → KARAR → SONUC → TEPKI → HAZIRLIK... 1. Event cikar (dunya degisir) 2. Kart cekersin (seceneklerin belirlenir) 3. Kart oynarsin (kararini verirsin) 4. Sistem calisir (sonuclari izlersin) 5. Rakip oynar (tepki alirsin) → Tekrar 1'e don

Run Yapisi (25 Tur = ~35-45 dk)

Asama 1: Kurulus (Tur 1-7)

Ilk isletme, temel calisanlar, para sikintisi. Oyuncu mekanigi ogrenir.

Asama 2: Buyume (Tur 8-15)

Combo'lar baslar, 2-3 isletme aktif, rakip agresiflesir, market share kavgasi.

Asama 3: Savas (Tur 16-22)

Buyuk combo'lar, market dominasyonu, yogun rakip baskisi, kriz eventleri.

Asama 4: Final (Tur 23-25)

Son hamleler, ekonomi kirilma noktasi. Ya MONOPOL ya IFLAS.

Analiz 1: Oyun Tasarimi Basarisizligi

Excel Problemi

Is temasi paradoks yaratti: spreadsheet seven insanlar zaten Capitalism Lab oynuyor. Deckbuilder seven insanlar "calisan ise al" gorunce "bu is gibi" dedi.

Ortadaki Olum Vadisi

Tur 1-5: hizli. Tur 6-10: eglenceli. Tur 11-20: olum vadisi. Tur 21-25: sonuc zaten belli olan formalite. Toplam 5 dakika gercek eglence.

Combo Hissettirilmiyor

Slay the Spire'da dusmanin cani eriyor. Empire of Cards'da dashboard'da sayi degisti. Oyuncu "tamam, bu guclu" deyip hicbir sey hissetmedi.

Analiz 2: Pazar Basarisizligi

Kimlik Krizi

"Deckbuilder" etiketi savas bekleyen oyunculari cekti. Isletme seven oyuncular etiketine hic bakmadi. Iki kitle arasinda kimseye ulasilamadi.

Capsule Art Felaketi

Dolar isaretli generic kart gorseli. 230x107 piksel thumbnail'de gorunmez. Mobil F2P oyundan ayirt edilemiyor.

%100 Dev, %0 Marketing

26 ay boyunca 11 tweet, 1 Reddit postu. Discord sunucusu lansmandan 3 hafta once. Kalite kendini satmaz.

Analiz 4: Teknik Basarisizlik

Combo Performans Bombasi

Sonsuz Para Spirali

Tur 15'te 800+/tur gelir vs 200/tur gider. Para anlamsizlasiyor. Gerilim buharlasiyor.

Save Corruption

EA'dan v1.0'a geciste kart ID'leri degisti. Oyuncularin %30'u meta progression kaybetti.
